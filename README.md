# Courier Route Optimizer



Calculating optimal shipment routes for couriers on a 2D plane, with randomly generated customer locations and starting points.



This application generates random customer locations and warehouse (depot) positions, assigns customers to the nearest warehouse using clustering, and calculates the optimal delivery route for each courier using a Nearest Neighbor algorithm.
